# Sirjanjeet Singh — Portfolio

Minimal, professional portfolio built with **Next.js 14** + **Tailwind CSS**.

## Getting Started

```bash
npm install
npm run dev
# http://localhost:3000
```

## Production Build

```bash
npm run build && npm start
```


## Resume
Place your resume PDF at `/public/resume.pdf` for the download button to work.

## Customization
- Colors → `tailwind.config.js` under `colors`
- Fonts  → `app/globals.css` `@import` + `tailwind.config.js` fontFamily
- Content → edit individual component files in `/components`

## Stack
- Next.js 14 (App Router)  
- Tailwind CSS  
- Google Fonts: Plus Jakarta Sans + IBM Plex Mono
